Datasets

Food 101

The Food-101 dataset is a challenging image dataset of 101 food categories, with 101,000 images in total: Number of images: 101,000 images, with 750 training images and 250 test images for each category.

Allergen100

The Allergen100 dataset contains the 100 most common food allergens along with their associated side effects.After categorizing the food, ingredients are identified using the recipe_w_search_terms dataset. Ingredients are then matched with potential allergens from the Allergen100 dataset.

recipe_w_search_terms

This dataset contains recipe metadata along with search queries that return the recipe and tags assigned by users to the recipe.The recipe_w_search_terms dataset has been pre-trained using the MobileNetV2 model.
